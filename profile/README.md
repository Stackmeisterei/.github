# 🇩🇪 Willkommen bei der `Stackmeisterei` - dem Solution Team Platform der HAFN IT

Das Solution Team Platform ist ein spezialisiertes Team innerhalb von HAFN IT, das sich auf die Entwicklung, Bereitstellung und den Betrieb von Cloud-nativen Plattformlösungen auf Basis von Microsoft Azure konzentriert.

## 🌐 Unsere Mission
Wir entwickeln skalierbare, sichere und automatisierte Plattformlösungen, die den Betrieb von Cloud-Anwendungen vereinfachen und beschleunigen. 
Unser Ziel ist es, Kunden durch standardisierte und dennoch flexible Architekturen optimal zu unterstützen.

## 🧩 Unsere Produkte

### 🔹 Solution Foundation Platform
Die Solution Foundation Platform ist unsere zentrale Plattformlösung, bestehend aus:
- **Cloud Platform Solution (CPS)**: Subscriptions für Plattformdienste wie Connectivity, Operations und Governance.
- **Cloud Application Solution (CAS)**: Subscriptions für Applikationsworkloads.

### 🔹 HAFN Best Practices
Unsere Deployments basieren auf den HAFN Best Practices und nutzen:
- Infrastructure-as-Code (Bicep)
- CI/CD Pipelines (GitOps, YAML)
- Automatisierung und Monitoring

## 🛠 Technologien
- Microsoft Azure
- GitHub Actions, Azure DevOps
- Bicep, Terraform, YAML
- ARM Templates

## 📦 Rollen im Team
- **Cloud Solution Architect (CSA)**
- **Cloud Solution Engineer (Engineering & Administration)**

# 🇬🇧 Welcome to `Stackmeisterei` - the Solution Team Platform at HAFN IT

The Solution Team Platform is a dedicated team within HAFN IT focused on building, delivering, and operating cloud-native platform solutions based on Microsoft Azure.

## 🌐 Our Mission
We build scalable, secure, and automated platform solutions to simplify and accelerate cloud application operations. Our goal is to support customers with standardized yet flexible architectures.

## 🧩 Our Products

### 🔹 Solution Foundation Platform
Our core platform offering includes:
- **Cloud Platform Solution (CPS)**: Subscriptions for platform services like connectivity, operations, and governance.
- **Cloud Application Solution (CAS)**: Subscriptions for hosting application workloads.


### 🔹 HAFN Best Practices
Our deployments follow HAFN best practices using:
- Infrastructure-as-Code (Bicep)
- CI/CD Pipelines (GitOps, YAML)
- Automation and Monitoring
